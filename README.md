# Perfect Udyog Samuha



## 📌 Project Overview

**Perfect Udyog Samuha** is a web-based application developed to manage business operations efficiently.  
The system provides an easy-to-use interface for handling organizational activities, user management, and business-related data in a streamlined way.

This project focuses on delivering a responsive UI, secure backend integration, and smooth data management functionality.

---

## 🚀 Features

- 🔐 User Authentication & Login System
- 📊 Interactive Dashboard
- 📝 Data Management System
- 🔍 Search & Filter Functionality
- 📱 Responsive User Interface
- 🔗 REST API Integration
- 💾 Database Connectivity

## ⚙️ Installation & Setup

### 1️⃣ Clone Repository

```bash
git clone https://github.com/vaishnavimhaske99/PerfectUdyogSamuha.git
```

### 2️⃣ Navigate to Project Folder

```bash
cd PerfectUdyogSamuha
```

---

## ▶️ Frontend Setup (Angular)

### Install Dependencies

```bash
npm install
```

### Run Angular Application

```bash
ng serve
```

Frontend runs on:

```bash
http://localhost:4200
```

---

## ▶️ Backend Setup (Spring Boot)

### Configure Database

Update database configuration in:

```properties
application.properties
```

Example:

```properties
spring.datasource.url=jdbc:mysql://localhost:3306/perfect_udyog
spring.datasource.username=root
spring.datasource.password=yourpassword
```

### Run Spring Boot Application

```bash
mvn spring-boot:run
```

Backend runs on:

```bash
http://localhost:8080
```
